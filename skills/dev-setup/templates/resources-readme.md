# resources/ — 素材置き場

ユーザーが用意した素材をここに配置する。
Claude はこのフォルダの内容を参照して実装に使用する。

---

## フォルダ構成

```
resources/
  images/        # 画像全般（PNG, JPG, SVG, WebP 等）
  icons/         # アイコン素材（SVG, ICO, PNG 等）
  fonts/         # カスタムフォント（.ttf, .woff, .woff2 等）
  design/        # デザインカンプ・参考画像（Figmaスクショ等）
  sounds/        # 効果音・BGM（あれば）
  other/         # 上記に当てはまらない素材
```

## 使い方

1. 上記フォルダに素材を配置する
2. Claudeに「resources/ にある〇〇を使って」と伝える
3. Claudeが `src/` へ自動でコピー or 参照して実装する

## 注意

- このフォルダの素材は **直接 src/ から参照しない**。必要な素材はビルドプロセスやコピー処理を通じて配置する
- 著作権・ライセンスに問題のある素材は配置しないこと
- 大容量ファイル（動画等）は Git LFS の利用を検討すること
